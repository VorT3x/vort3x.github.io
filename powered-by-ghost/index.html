Powered by Ghost

May 16, 2015

Hello World!

I wanted to start blogging years ago and it’s time to start. I’ve chosen Ghost for my blog, hosted it on DigitalOcean and bought template made by EckoThemes from ThemeForest.

Mostly I’ll write technical articles. Currently I feel real passion to Node.js and want to dig it further, that’s one of the reasons why I’ve chosen Ghost for blogging, since it’s built on Node.js. I really like how simple it’s comparable to Wordpress and it uses Markdown syntax for posts, which is also pleasant to use.

Stay tuned!
